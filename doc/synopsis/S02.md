Bits and Pieces
===============

## Variable

### Integer, Float and Double

    foo = 123
    foo2 = 1.23

### String

    bar = '123'

### String interpolation

    person = 'John'
    template = "Hi, {{ person }}. Good morning."

will be transformed into:

    <?
    $person = 'John';
    $template = 'Hi, ' . $person . '. Good morning.';

## Hash (Associative Array)

    list = 1..10

    $list = range(1,10);

    list = [1,2,3]
    list[] = 4
    list[] = 5
    list[] = 6


    zoo = "hack"
    addresses = {
        "foo" => 1,
        :bar => 2,
        zoo  => 3,
    }
    addresses[:john] = 3
